Lab Goals

Activities that incorporate accessibility education and improve understanding of accessible design are available.

Learn accessible design

Learn the concepts that you need to follow to understand accessible code development and design.

Design For Visually Impaired

Visual Impairments cause issues in use of the web, for those affected. Learn how to make websites that they can use.

Design For Dexterity Issues

Dexterity Issues cause erroneous input to be entered by the user. Learn the guidelines you need to know to make it possible for those affected to use the web.
